# Theme Palettes

Canonical local path for the Emberstone dark palette:

- `~/.config/themes/palettes/emberstone-dark.json`

The palette is versioned directly in this repo:

- `dotfiles/themes/palettes/emberstone-dark.json`

Environment variables exported in `~/.config/environment.d/90-theme.conf`:

- `THEME_PALETTE_DIR`
- `THEME_PALETTE_DARK`
- `THEME_PALETTE_EMBERSTONE_DARK`
